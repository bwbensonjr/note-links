Frequently asked questions
--------------------------

Everything you need to know about Site Spy

What is Site Spy?

Site Spy is a website monitoring tool that tracks pages for changes automatically. It checks on a schedule you set, detects content updates, and notifies you instantly. Use the web dashboard on any device, or add the browser extension for one-click tracking in Chrome and Firefox.

How does website change detection work?

Site Spy takes snapshots of the pages you track at your chosen interval (from minutes to weeks). When new content is detected, it highlights exactly what changed using a visual diff view — additions appear in green and removals in red. You can browse the full snapshot timeline to compare any two versions.

Is Site Spy free to use?

Yes, Site Spy has a free plan that lets you track up to 2 websites with checks every hour. All notification channels (email, Telegram, push, webhooks) are included free on every plan. Paid plans unlock more watches, faster check intervals, and advanced features like AI summaries and visual diff.

How can I access Site Spy?

The web dashboard works in any browser, including mobile. For desktop, you can also install the browser extension for Chrome or Firefox. Your watches sync across all devices when you sign in.

Can I track specific parts of a page instead of the whole page?

Yes. Site Spy includes an element inspector that lets you click on any element on a webpage to monitor just that section. This is useful for tracking price changes, stock availability, news headlines, or any specific content block.

What are AI smart filters?

When you add a website, describe what you care about in plain English — like 'price changes', 'new job listings', or 'when reservations open'. Site Spy's AI analyzes the page and automatically configures filters to ignore noise like timestamps, cookie banners, visitor counters, and dynamic ads. You only get notified when something meaningful changes. No CSS selectors or manual setup required.

How is Site Spy different from Visualping or Distill?

Most website monitoring tools make you manually pick CSS selectors or draw boxes around page areas. Site Spy uses AI to understand what you care about in plain English and auto-configures filters for you. It also includes a built-in AI chat assistant, visual diffs, MCP integration for AI agents, and a browser extension for Chrome and Firefox — all with a free plan.

Does Site Spy work with AI assistants?

Yes. Site Spy provides an MCP (Model Context Protocol) server that integrates with Claude, Cursor, and other MCP-compatible AI agents. Your AI assistant can monitor websites, compare snapshots, and summarize changes directly in your chat. Site Spy also has a built-in AI chat where you can set up monitors, configure filters, and check what changed through natural conversation.
