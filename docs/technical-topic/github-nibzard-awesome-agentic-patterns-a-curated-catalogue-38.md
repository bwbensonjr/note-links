# GitHub - nibzard/awesome-agentic-patterns: A curated catalogue of awesome agentic AI patterns

Awesome Agentic Patterns A curated catalogue of agentic AI patterns — real‑world tricks, workflows, and mini‑architectures that help autonomous or semi‑autonomous AI agents get useful work done in production. Why? Tutorials show toy demos. Real products hide the messy bits. This list surfaces the repeatable patterns that bridge the gap so we can all ship smarter, faster agents. What counts as a pattern? Repeatable – more than one team is using it. Agent‑centric – improves how an AI agent senses, reasons, or acts. Traceable – backed by a public reference: blog post, talk, repo, or paper. If your link ticks those boxes, it belongs here.
