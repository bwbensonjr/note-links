Chroma Explorer – Desktop Client for ChromaDB
=============================================

[Powered by![ChromaDB logo](/images/chroma-logo.svg)](https://www.trychroma.com)

Chroma Explorer is a native macOS desktop application designed for working with ChromaDB vector databases. Easily browse and manage your collections, documents, embeddings, and metadata in an intuitive interface. Run semantic search queries and view similarity scores to understand how your data is organized. Whether you're debugging vector embeddings or exploring stored data, Chroma Explorer provides the tools you need to inspect and manage your ChromaDB instances efficiently.

A modern, native desktop client for ChromaDB. Browse collections, search semantically, and manage your vector embeddings with ease.

Explore ChromaDB Collections and Documents
------------------------------------------

Everything you need to work with ChromaDB, in a beautiful native app.

### Multi-Profile Connections

Connect to local, remote, or Chroma Cloud databases. Save and manage multiple connection profiles with secure API key storage.

### Collection Management

Create, copy, and configure collections with ease. Set custom embedding functions and HNSW parameters.

### Semantic Search

Search your documents using natural language. Find similar content instantly with vector similarity search.

### 13+ Embedding Providers

Built-in support for OpenAI, Cohere, Gemini, Ollama, Jina, Mistral, Voyage AI, and more.

### Document Operations

Browse, create, edit, and delete documents. Batch operations for efficient bulk document management.

### Native macOS Experience

Beautiful glass morphism design that feels right at home on your Mac.
