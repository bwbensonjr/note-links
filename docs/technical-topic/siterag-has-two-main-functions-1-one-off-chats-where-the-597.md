# SiteRAG has two main functions:

1) One-off chats where the content is stuffed into the model's context
2) RAG over pages/sites, where the page contents are chunked, embedded and stored for semantic search. This mode also has an advanced "multi-query" option where it will generate multiple queries from your initial input, then preform semantic search over them all.

Since it's built with LangChainAI, it supports multiple LLM providers: OpenAI, Anthropic, DeepSeek, Gemini, etc
