# The Logfile Navigator | The Logfile Navigator, lnav for short, is an advanced log file viewer for the small-scale.

A log file viewer for the terminal Merge, tail, search, filter, and query log files with ease. No server. No setup. Still featureful. Try it out: $ ssh playground@demo.lnav.org Easy to Use Just point lnav at a directory and it will take care of the rest. File formats are automatically detected and compressed files are unpacked on the fly. Online help and previews for operations make it simpler to level up your experience.
