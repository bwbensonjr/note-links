# CFRS[] Community Demos

CFRS[] Community Demos This page contains a collection of CFRS[] demos contributed by community members. CFRS[] is an extremely minimal drawing language that consists of only six simple commands: C , F , R , S , [ and ] . To learn more about CFRS[], visit susam.net/cfrs.html or read the manual at github.com/susam/cfrs . If you create an interesting demo that fits in 64 bytes of code, please create a new post here and share it with the community. If the demo looks interesting, it may be added to this page.
