# Categorization Tags

Tags are automatically assigned by an LLM-based tagger that analyzes link titles, summaries, descriptions, URLs, and domains.

## Programming Language

Tags for content primarily about or using a specific programming language.

- `python` - Python programming, Django, Flask, FastAPI, pandas, etc.
- `rust` - Rust programming, Cargo, crates.io
- `typescript` - TypeScript, Deno
- `javascript` - JavaScript, Node.js, React, Vue, Angular
- `lisp` - General Lisp family content
- `common-lisp` - Common Lisp specifically (SBCL, CCL)
- `clojure` - Clojure/ClojureScript
- `scheme` - Scheme, Racket, Guile
- `haskell` - Haskell programming
- `go` - Go/Golang
- `c` - C programming
- `cpp` - C++ programming
- `nix` - Nix, NixOS, nix-darwin, flakes
- `sql` - SQL, database queries
- `swift` - Swift, iOS/macOS development
- `java` - Java, JVM (not Clojure)
- `ruby` - Ruby, Rails
- `elixir` - Elixir, Phoenix, Erlang VM
- `zig` - Zig programming

## Technical Topic

Tags for technical content that isn't language-specific.

- `ai` - Artificial intelligence, machine learning, neural networks
- `llm` - Large language models, ChatGPT, Claude, prompt engineering
- `compilers` - Compilers, interpreters, parsing, lexing, ASTs, LLVM
- `github-repo` - GitHub repository links
- `database` - Databases, SQL, PostgreSQL, SQLite, data storage
- `devops` - Infrastructure, deployment, CI/CD, containers, Kubernetes
- `web-dev` - Web development, HTML, CSS, frontend
- `academic-paper` - Research papers, arxiv, academic content
- `tutorial` - How-to guides, learning content, courses
- `cli-tool` - Command line tools, terminal utilities
- `distributed-systems` - Streaming, Kafka, distributed computing
- `security` - Cybersecurity, privacy, encryption
- `emulator` - Emulators, virtual machines, QEMU

## Culture

Tags for non-technical content.

- `tv` - TV shows, television series, streaming shows
- `movie` - Films, cinema
- `fiction-book` - Novels, fiction, short stories
- `nonfiction-book` - Nonfiction, biography, memoir, essays
- `music` - Music, albums, artists, Spotify content
- `news` - Current events, journalism
- `politics` - Political content
- `podcast` - Podcasts, audio content
- `video` - YouTube videos, video content
- `gaming` - Video games, gaming
- `social-media` - Threads, Twitter/X, Bluesky, social posts
